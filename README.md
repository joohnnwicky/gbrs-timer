# GBRS TIME CLOCK

一款日式简约风格的桌面拨盘番茄钟计时器应用。

![Preview](assets/icon.png)

## 功能特性

- **圆环计时器**：点击圆环任意位置开始倒计时，弧线从点击位置顺时针填充
- **自定义时长**：点击位置决定倒计时时长（最长60分钟）
- **今日累计**：显示今日完成的番茄钟总时长
- **历史记录**：查看每天的番茄钟使用记录
- **自定义提示音**：可选择自己的mp3文件作为完成提示音
- **悬浮窗口**：小巧的窗口始终置顶，不影响工作

## 技术栈

- Electron 28
- Canvas 2D API（圆环绘制）
- electron-store（数据持久化）

## 安装使用

### 直接运行exe

下载 `dist/win-unpacked/GBRS TIME CLOCK.exe` 直接运行。

### 开发模式

```bash
# 安装依赖
npm install

# 运行开发版本
npm start

# 打包生成exe
npm run build
```

## 使用说明

1. **点击圆环**：点击任意位置开始倒计时，时长由点击位置决定
2. **暂停/继续**：点击暂停按钮可暂停计时
3. **重置**：点击重置按钮清空当前计时
4. **提示音**：点击"提示音"按钮选择自定义mp3文件
5. **历史**：点击历史按钮查看使用记录
