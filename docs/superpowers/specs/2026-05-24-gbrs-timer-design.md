# GBRS TIME CLOCK - 拨盘番茄钟设计文档

## 项目概述

一个日式简约风格的桌面番茄钟计时器，以圆环形式呈现。用户点击圆环任意位置开始倒计时，一圈代表60分钟，红色填充随时间流逝逐渐"消融"。

## 技术方案

**框架：Electron + Canvas**

- Electron 提供桌面应用框架和悬浮窗口能力
- Canvas 2D API 绘制圆环动画
- HTML/CSS 实现日式简约风格界面
- 本地 JSON 存储历史记录

## 一、界面布局

### 窗口规格

| 属性 | 值 |
|------|-----|
| 尺寸 | 280×320 像素 |
| 形态 | 圆角矩形，无系统标题栏 |
| 位置 | 始终置顶悬浮 |
| 背景 | 米白色 `#F5F5F0` |

### 布局结构

从上到下：

1. **标题区** - "GBRS TIME CLOCK"，黑色斜体，居中
2. **累计区** - "今日累计：XXX分钟"，细体字
3. **圆环区** - 红色填充圆环，中心显示剩余时间 `MM:SS`
4. **控制区** - [历史] [暂停] [重置] 按钮

### 圆环设计

- 外径：160px，内径：120px（环宽 40px）
- 颜色：温暖红 `#D4534A`
- 填充：从点击角度开始，顺时针方向
- 消融效果：弧线末端柔和渐变淡出

### 字体

- 标题：`Georgia Italic` 或斜体衬线字体
- 数字：`Roboto Mono Light` 或纤细等宽字体

## 二、核心交互

### 启动倒计时

1. 用户点击圆环任意位置
2. 计算点击角度（0°=顶部，顺时针方向）
3. 角度转分钟：`angle/360 * 60`
4. 红色弧线从该位置开始填充整圆
5. 倒计时开始，弧线顺时针"消融"回到起点

### 倒计时过程

- 中心显示剩余时间 `MM:SS`
- 弧线每秒更新，末端渐变淡出
- 可点击"暂停"暂停计时

### 倒计时结束

- 播放轻柔提示音（可配置静音）
- 圆环变空心
- 自动记录本次时长到今日累计
- 中心短暂显示"完成 ✓"

### 控制按钮

| 按钮 | 功能 |
|------|------|
| 历史 | 打开历史记录窗口 |
| 暂停 | 暂停当前倒计时 |
| 重置 | 清空当前倒计时，圆环恢复空心 |

- 保持日式简约风格

## 四、技术架构

### 项目结构

```
GBRS-Timer/
├── main.js                 ← Electron主进程
├── preload.js              ← 预加载脚本
├── package.json
├── assets/
│   ├── icon.png
│   └ sound.mp3
├── renderer/
│   ├── index.html
│   ├── styles.css
│   ├── timer.js            ← Canvas圆环绘制
│   ├── interaction.js      ← 点击交互控制
│   ├── storage.js          ← 数据读写
│   ├── history.html
│   └── history.js
```

### 关键技术点

| 功能 | 实现 |
|------|------|
| 圆环绘制 | Canvas `arc()` |
| 点击角度 | MouseEvent + 三角函数计算 |
| 动画 | `setInterval` 每秒更新 |
| 悬浮置顶 | `BrowserWindow` 配置 |
| 数据存储 | `electron-store` 或 `fs` |

### 依赖

- `electron` - 核心框架
- `electron-store` - 本地存储（可选）
